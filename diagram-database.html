Diagrama de Base de Datos

Sistema de Gestión de Usuarios y Roles

Usuarios
id UUID (PK)
nombre VARCHAR(100)
correo VARCHAR(150) UNIQUE
password VARCHAR(255)
activo BOOLEAN
rol_id UUID (FK)
created_at TIMESTAMP
updated_at TIMESTAMP
Roles
id UUID (PK)
nombre VARCHAR(50) UNIQUE
descripcion TEXT
activo BOOLEAN
created_at TIMESTAMP
updated_at TIMESTAMP
UserProfiles
id UUID (PK)
userId UUID (FK) UNIQUE
bio TEXT
avatarUrl VARCHAR(255)
created_at TIMESTAMP
updated_at TIMESTAMP
UserRoles
id UUID (PK)
userId UUID (FK)
roleId UUID (FK)
created_at TIMESTAMP
updated_at TIMESTAMP

Relaciones

1:N - Usuario → Rol
Un usuario tiene un rol principal (rol_id), un rol puede tener muchos usuarios
1:1 - Usuario ↔ UserProfile
Cada usuario puede tener un perfil extendido con información adicional
N:M - Usuario ↔ Rol
Un usuario puede tener múltiples roles, un rol puede ser asignado a varios usuarios (tabla UserRoles)
