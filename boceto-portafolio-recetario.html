Mi Comprensión

Buenas Prácticas que Aplico

Mis Buenas Prácticas

1 Calidad sobre cantidad: Prefiero mostrar 3-5 proyectos bien documentados que 20 sin contexto. Cada proyecto en mi portafolio cuenta una historia clara: el problema que resolví, cómo lo hice, qué tecnologías usé y qué aprendí. Además, incluyo explicaciones pedagógicas para que otros puedan aprender.

2 Actualización constante: Mantengo mi portafolio vivo y en crecimiento. Agrego nuevos proyectos del bootcamp, mejoro los existentes con mejor documentación y elimino trabajos que ya no representan mi nivel actual. Esto refleja mi compromiso con el aprendizaje continuo.

3 Claridad y enfoque pedagógico: Mi portafolio debe ser fácil de navegar y educativo. Uso estructura lógica, títulos claros, descripciones concisas y me aseguro de que funcione en móviles. Mi formación pedagógica me ayuda a presentar información de manera didáctica y accesible.

Mi Elección

Herramientas para Mi Portafolio

GitHub Pages

Por qué lo elegí: Gratuito, integrado con Git, ideal para desarrolladores y perfecto para mostrar código

Elegí GitHub Pages porque me permite alojar mi portafolio directamente desde mi repositorio. El proceso fue simple: creé el repositorio, subí mis archivos HTML/CSS/JS, activé GitHub Pages, y ahora está disponible en magdaig.github.io/portafoliojs/. Esto me permite demostrar mis habilidades técnicas mientras mantengo el código accesible para otros estudiantes.

HTML/CSS/JavaScript

Por qué lo elegí: Control total, aprendizaje profundo, personalización ilimitada

Construí mi portafolio con HTML/CSS/JavaScript desde cero porque me da control total sobre el diseño y funcionalidad. Usé HTML semántico para el contenido, CSS con variables para mantener consistencia, y JavaScript para interactividad. Integré Bootstrap para componentes responsive y AOS para animaciones. Esta elección demuestra mis habilidades frontend y me permite personalizar completamente la experiencia educativa.

Mi Elección Final

GitHub Pages + HTML/CSS/JavaScript

Mis Razones: Esta combinación es perfecta para mí porque demuestra mis habilidades técnicas directamente, me da control total para crear una experiencia educativa personalizada, se integra perfectamente con Git para versionado, es completamente gratuito, me permite aprender continuamente mientras mantengo y mejoro el portafolio, y el código es portable a otros servicios si es necesario. Además, al ser código abierto, otros estudiantes pueden aprender de mi trabajo.
